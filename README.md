# 🔍 Subdomain Finder (Python GUI Tool)

A simple and user-friendly Python-based GUI tool to discover subdomains of a given domain using public certificate transparency logs.

## 🚀 Features

* Clean and modern GUI (Tkinter-based)
* Fast subdomain enumeration
* Uses crt.sh public data (no API key required)
* Scrollable output for better readability
* Threaded scanning (no UI freeze)

## 🛠️ Tech Stack

* Python 3
* requests
* tkinter

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/your-username/subdomain-finder.git
```

2. Navigate to the project folder:

```
cd subdomain-finder
```

3. Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Usage

Run the tool:

```
python subfinder_gui.py
```

Enter a domain (e.g., example.com) and click on **"Find Subdomains"** to start scanning.

This tool is intended for educational and authorized testing purposes only. Do not use it on systems without proper permission.
